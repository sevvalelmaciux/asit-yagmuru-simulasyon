# 🌧️ Asit Yağmuru Simülasyonu

## 📌 Proje Amacı
Bu proje, SO₂ (kükürt dioksit) ve NOₓ (azot oksitler) gazlarının asit yağmuruna etkisini görselleştirmek için hazırlanmıştır.

## ⚙️ Nasıl Çalışır?
Kullanıcı emisyon değerlerini değiştirerek pH seviyesindeki değişimi gözlemler. pH düştükçe çevresel etkiler artar.

## 🧪 Sonuç
Simülasyon, insan faaliyetlerinin çevre üzerindeki etkisini görsel olarak ortaya koymaktadır.
