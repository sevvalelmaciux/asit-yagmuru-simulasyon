# asit-yagmuru-simulasyon
Asit yağmuru; atmosferde bulunan kükürt dioksit (SO₂) ve azot oksitler (NOₓ) gibi gazların su buharı ile tepkimeye girerek asidik bileşikler oluşturması sonucu meydana gelir. Bu proje, bu gazların miktarına bağlı olarak yağmurun pH değerinin nasıl değiştiğini ve çevre üzerindeki etkilerini görselleştirmek amacıyla hazırlanmıştır.
